Introduction

This is the simple page that shows a number of bootstrap components and the styles they provide to enhance plain HTML content- while also supporting a responsive display ( screen resizing).

Firstly, at the top of the page , within a section, is a navigation list using the Bootstrap component. Next is the section that contains a banner ( a little of the page and leading sentence, all with a blue background), and the content of the page. The content contains some pretty pictures of trees ( using different layouts), as well as a bootstrap formatted table with content. The final section is a footer.

Tree Pictures(Thumbnails)

Below are some pretty pictures of trees, downloaded from the website Unsplash, with beautiful and freely-usable images. These images are placed in a single bootstrap row composed of three columns ( using a combined col-sm-4 col-12 ,where the col-sm-4 forces three images to be in a row on all the screen sizes except very small ones (576px) where the col-12 forces the images to use all the 12 column grid units and stack each image on top of each other). Each image has the class img-thumbnail applied to it, allowing it resize to the size of the column.

Tree Pictures with Text

The following image and text is held within a new div which has the class attributes p-4 mb-4 bg- light rounded, to set the padding margin-below , background color and corner shaped, respectively. It holds a row, with the image is assigned to a col-sm-4 column and the text assigned to a single col-sm-8; allowing it to span the remaining 8 column units. Added to each of these two column is the col-12, which says when the screen is smaller than 576px, it will then use all the 12 columns grids for each image and text, allowing them to be stacked on top of each other. In the final row, as with the one above, within a new div block we first have the text column has been assigned using a col-sm-8 col-12, followed by the image held in a column assigned a class col-sm-4 col-12; reversing the order shown in the previous row.
